새 사용자 추가
@userinfobot을 통해 사용자 ID를 확인할 수 있습니다.
이름을 입력하지 않으면 "사용자{ID}"로 자동 설정됩니다.
팁: 사용자를 추가한 후에는 해당 사용자가 텔레그램에서 봇에게 /start 명령어를 전송해야 메시지를 받을 수 있습니다.
취소
사용자 ID 확인 방법
  1. 텔레그램에서 @userinfobot을 찾습니다
  2. 봇에게 아무 메시지나 전송합니다
  3. 봇이 응답으로 보내는 숫자가 사용자 ID입니다
  4. 해당 ID를 위의 입력란에 입력하세요
주의: 사용자 ID는 숫자여야 하며, 음수일 수도 있습니다 (그룹 채팅의 경우).
